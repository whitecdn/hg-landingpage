# HostGenius Landing Page

This is a high-converting, "dark premium" landing page built with Next.js and Tailwind CSS, based on the HostGenius Notion 1-pager.

## Features
- **Sleek Aesthetic**: Dark mode by default, glassmorphism, and neon accents.
- **Interactive**: Framer Motion animations for smooth entrance and hover effects.
- **Optimized**: Fast loading, SEO ready.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser.

## Deployment

This project is ready to be deployed on Vercel.
1.  Push to GitHub.
2.  Import project in Vercel.
3.  Deploy.
